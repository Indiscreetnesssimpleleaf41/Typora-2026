## 📝 Using the editor

Once you install Typora-2026, you find a shortcut icon on your Windows desktop. Click this icon to open the program. The screen shows a blank page. You can type text here just like a word processor. 

Markdown provides a way to format text as you write. You use specific symbols to create headers, bold text, or lists. Because this editor uses live preview, you see the bold text become thick immediately after you type the markers. 

To create a header, type the hash symbol (#) followed by a space. You can use multiple hash symbols to create smaller subheaders. Use one hash for the title and two or three for sections.

To make text bold, put two asterisks before and after your word. For example, typing **bold text** makes the words appear heavy on the screen. To make text italic, put one asterisk before and after the word. 

## 📂 Organizing your work

Typora-2026 saves your work as a .md file. This file type works with most text editors. You can move these files between folders on your computer. You can also upload them to cloud storage services like OneDrive or Google Drive.

To open an existing file, select "Open" from the "File" menu. Browse your folders to find the document you want to edit.
